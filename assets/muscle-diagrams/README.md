# Muscle Diagram Images

This folder should contain muscle diagram images for each machine showing which muscles are engaged.

## Required Images

For each machine, add a PNG image showing the muscles worked:

1. `leg_press.png` - Leg Press muscle diagram (quadriceps, glutes highlighted)
2. `lat_pulldown.png` - Lat Pulldown muscle diagram (latissimus dorsi highlighted)
3. `chest_press.png` - Chest Press muscle diagram (pectoralis major highlighted)
4. `seated_row.png` - Seated Row muscle diagram (middle back, rhomboids highlighted)
5. `shoulder_press.png` - Shoulder Press muscle diagram (deltoids highlighted)

## Where to Get Images

You can find muscle diagram images from:

1. **Free Resources:**
   - Search "muscle anatomy diagram" on sites like Unsplash, Pexels
   - Use open-source anatomy illustrations from websites like:
     - https://www.bodybuilding.com (exercise database)
     - https://exrx.net (exercise directory with muscle diagrams)

2. **Create Your Own:**
   - Use tools like Canva or Figma with anatomy templates
   - Highlight primary muscles in red/dark color
   - Highlight secondary muscles in orange/lighter color

3. **Purchase:**
   - Stock photo sites (iStock, Shutterstock)
   - Fitness app asset packs

## Image Specifications

- Format: PNG with transparency
- Size: 400x400px to 800x800px
- Primary muscles: Highlighted in red/dark color (#FF0000 or similar)
- Secondary muscles: Highlighted in orange/light color (#FF9800 or similar)
- Background: Transparent or white

## For MVP Testing

For now, the app will show a placeholder text where the diagram should appear. You can test the app without these images, but they greatly enhance the user experience.
